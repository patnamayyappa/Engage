Azure Functions

Values in Configuration Keys
  1. Log on to https://portal.azure.com and click Function Apps in the left pane.
  2. Select the App Service and then click Application Settings.

    Ensure that values are specified in the following configuration keys:ClientId, ClientSecret, CrmOrgUrl, TenantId.
  3. To verify scheduled time in Azure functions, note the key named RetentionScoreCalculatorSchedule with the value 0 0 */8 * * * (indicates that it runs every 8 hours).
    The schedule is specified using the on CRON format.

Application User Credentials

At least one application user must be created with Application ID details that are identical to the value in the ClientId field.

  • On the Users page, select the Application Users view.
    Verify that the value in the Application ID field is identical to the value in the ClientId field on the Azure portal.
