Creating a DOM Definition Execution Order

DOM Definition execution order records determine the order in which the logic defined in the associated DOM Definition Logic record runs.

Ensure that values are selected or typed appropriately in the following fields:

  • In the Order field, type a numeric value which determines the order in which the logic defined for the DOM definition will run.
    Multiple DOM Definition Execution Order records for the same DOM Master will execute in ascending order based on the value in the Order field.
