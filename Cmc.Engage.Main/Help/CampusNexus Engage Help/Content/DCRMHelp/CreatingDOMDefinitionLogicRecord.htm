Creating a DOM Definition Logic Record

DOM Definition Logic records include logic based on which records in the associated DOM Master will be assigned.

Ensure that values are selected or typed appropriately in the following fields:

  • In the Attribute field, a custom interface is used to set its value.
  • In the Condition Type field, select one of the following values:
    • Begins With
    • Equals
    • Not Null
    • Null
    • Range
    • Record Owner
  • The following fields are enabled or disabled based on the selected value:
    • Value
    • Minimum
    • Maximum

    To complete the process of defining logic to assign records, type appropriate content in these fields based on the type of the field specified in the Attribute field.
