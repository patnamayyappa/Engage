Creating a Lifecycle

A lifecycle is created for a contact based on the information available in the Inbound Interest using one of the following methods:

  • Automatically qualify contacts
  • Manually create a lifecycle for a Contact

Automatically Qualify Contacts

If a Contact meets the required qualification which is obtained from the details available in the Inbound Interest section, it is automatically qualified using the Inbound Interest – Create Lifecycleworkflow and a Lifecycle record is created and associated with the Contact.

For information on the Inbound Interest – Create Lifecycle workflow, see Auto-Qualifying an Inbound Interest Record.

Manually Add a Lifecycle for a Contact

Sometimes, the information in the Inbound Interest record associated with a Contact is not sufficient to automate the process of creating a Lifecycle. In such scenarios, user intervention is needed to create a Lifecycle record for an Inbound Interest record of a Contact.
