Creating or Editing a Staff Survey Template

A staff survey template is created by defining a list of questions to evaluate the performance of students who are pursuing a course in a school or university. Staff survey template(s) can be created and edited by an administrator.

Steps to create or edit a Staff Survey Template

  1. Go to Staff Survey Template.

  2. If you want to:

    • Create, click New.

    • Edit, select a Staff Survey Template from the list view.

  3. In the General section, specify or update values in the following fields:

    1. Staff Survey Template NameName of the Staff Survey Template.

    2. OwnerThe owner of the Staff Survey Template.

  4. In the Start Date and Due Date sections, select a value in the Calculation Type field to determine a static or calculated value for the start and end date of the Staff Survey.

    The following table describes the values of the fields depending on the selection in the Calculation Type field.

    When Static is selected:
    Start Date Static Select a date when the Staff Survey must begin.
    Due Date Static Specify the date by when the Staff Survey must complete.
    When Calculated is selected
    Number of Days Specify the number of days that will be used to calculate the start and due date of the Staff Survey.
    Days Direction

    Select After or Before to calculate the Start Date and Due Date fields.
    Calculation Field

    Select one of the following values to calculate the Start Date and Due Date fields:

    • Assignment Date - The date when a Staff Survey is assigned to an instructor.
    • Start Date of Academic Period- The start date of the Academic Period.
    • End Date of Academic Period - The end date of the Academic Period.

  5. In the Description section, add content that will be displayed as header text for the template. For example, university name/logo/website, explanatory text about the survey, etc.

  6. Click Save.

  7. Click the Designer option from the drop-down list as shown in the following image.

    The Staff Survey Template:Designer form is displayed that provides options to add questions and preview the template.

  8. In the Questions section, add the required questions and save the Staff Survey Template.

    The Staff Survey Template record is created and appears in the Staff Survey Templatelist view.

Note

The student names or contacts in a Staff Survey are populated from the Staff Course that is added to the Course History which in turn is added to Contacts.
