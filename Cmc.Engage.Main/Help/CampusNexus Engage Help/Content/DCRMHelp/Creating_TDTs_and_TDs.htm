Creating Success Plans and To Dos

Success Plans and To Dos are nearly identical to success plan templates and success plan to do templates. The processes to create the records are similar, but are not exactly the same. There are a few different fields between the entities, and the entities also serve two different purposes. For more information, see About Success Planning.

The difference occurs when a success plan template is assigned to a student. A copy of the assigned success plan template record is automatically created within the student's record. This copied record is a success plan record. The name of the success plan is suffixed with the name of the student. However, if a success plan is created directly from the success plan grid or within a student's record, the value in the name field does not follow this convention.

Note

When you create a success plan record, you will be required to specify information in the following mandatory fields:

  • Success Plan Name
  • Assigned to Student
  • Status: by default, the value Active is set in this field.
  • Owner: by default, your name is set. You can set a different owner.

On saving the success plan record, you can create a To Do record. When you view the new To Do form from within a success plan, the following fields are updated by default:

  • Student Can Complete: By default, the value Yes is set. Click in the field to set the value to No. If the value is No, the advisor must complete the activity in the To Do.
  • Assigned to Staff: By default, your name is set. You can set a different staff member.
  • Success Plan: By default, the parent success plan is displayed. You can set a different success plan.
    Additionally, the following fields are important:
    • Ownership Type: It can have one of the following values:
      • Staff Owned
      • Student Owned
    • Required/Optional: Indicates if the To Do must be completed or is optional
    • Due Date: Indicates the date when the To Do must be completed. For more information about this field, see Creating Success Plan Templates and Success Plan To Do Templates.
