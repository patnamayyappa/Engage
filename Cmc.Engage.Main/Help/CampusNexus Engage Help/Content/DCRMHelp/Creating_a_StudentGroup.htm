Creating a Student Group

When you create a student group, select values in the following fields appropriately:

  • List Type: select from the following values
    • Static – the list of students in the group is unchanged without user intervention in the duration of the group.
    • Dynamic – students are added to the group dynamically. As long as the group is active and its expiration date is in the future, students will be added to the group if they fulfill conditions specified in the filter.
  • Marketing List Type – select the value Student Group.
  • Targeted At – select the value Contact.
  • Expiration Date – select an appropriate date.

After saving the record, you can add students to the group in the Members area.


Note

  • You can manually deactivate a student group record. Alternatively, student group records are deactivated automatically after the expiration date lapses.
  • If you activate an inactive student group, you’ll need to specify a new expiration date.
