Creating a Success Network Assignment Record

When you create a success network record, you are creating the association of a student group with a user who will be assigned a role that is selected in the Role field. Typically, a user is assigned the role of a mentor, coach, instructor, and so on. The user is required to work with the students in the group to help them complete their activities. For information about student groups, see About Student Groups.

Success network details are displayed in the Current Student Profile area of a student's record. The success network record is displayed in the format <student name> - <staff member>.

Note

You can also create a success network in a student record's Current Student Profile area. The created success network record will be applicable only to that student.
