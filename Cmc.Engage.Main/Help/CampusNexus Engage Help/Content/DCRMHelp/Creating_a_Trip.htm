Creating a Trip

A recruiter creates a trip to plan and conduct student recruitment in a university. It's essentially a framework or container to store a series of multiple subsidiary trips or events (referred to as trip activities) that are included in the parent trip.

Creating a trip record is basically a 3-step process where you, as the recruiter, are required to:

  1. Create the trip record
  2. Create trip activities in the record
  3. Seek approval for the trip

Creating a Trip

On specifying values in mandatory and other fields, when you save a trip record, the value in the Trip Status field is set to Planning. You are then required to add staff members and volunteers to the record.

Note

Bing map information of saved trip activities is displayed in the Map View area. It serves as a visual indicator to help review and optimize your travel plans.
